🚀 MERN Stack CRUD Application
A full-stack web application built with the MERN (MongoDB, Express.js, React, Node.js) stack to perform Create, Read, Update, and Delete (CRUD) operations on user data.

📋 Features
Create: Easily add new users to the database.

Read: View a dynamic list of all registered users.

Update: Modify existing user details seamlessly.

Delete: Remove users from the system with a confirmation flow.

Toast Notifications: Real-time feedback using react-hot-toast.

Responsive UI: Styled for a clean user experience.

🛠️ Tech Stack
Frontend: React, Axios, React Router, React Hot Toast

Backend: Node.js, Express.js

Database: MongoDB (Mongoose)

Styling: [e.g., CSS/Bootstrap/Tailwind]

⚙️ Installation & Setup
Prerequisites
Node.js (v14 or higher)

MongoDB (Local or Atlas URI)

1. Clone the repository
Bash
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
2. Setup Backend
Bash
cd backend
npm install
# Create a .env file and add your MONGO_URL and PORT
npm run dev
3. Setup Frontend
Bash
cd ../frontend
npm install
npm start
